# Summary

Simple mount for implement BLtouch into N2/N2+
Firmware mod welcome to activate BLtouch for N2/N2+
https://www.facebook.com/sylvain.blt/videos/1821384614746321/
Instructions:
1- install the BLTouch as described http://forum.raise3d.com/viewtopic.php?f=4&t=1030&start=20

2-upgrade the firmware (as a gcode) 

3- set the Z offset (gap between probe and nozzle) with appropriate  gcode into the zip file
BE CAREFUL :
Negative value decrease the distance between the bed and nozzle
Positive value increase the distance between the bed and nozzle
initial value is 0
4- insert the start gcode in the txt file to your slicer
